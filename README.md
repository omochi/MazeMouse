MazeMouse
=========

これ面白そうだから俺も書いてみる→ https://bitbucket.org/tokoro10g/mousesim

ビルド
=====

~~~
$ make
~~~

実行
====

## 迷路ビューワ

~~~
$ ./MazeView data/maze1.dat data/maze2.dat ...
~~~

## 迷路走行

~~~
$ ./ManualMouse [-a] data/maze1.dat
~~~

実行結果がlogディレクトリに吐かれます。
-aオプションを付けると自動走行します。
走行結果はログを見ると良い。
